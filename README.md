# agriculture-ui
UI for back-end project
## 目录结构
```
─resource
    ├─static
    │  ├─css
    │  │      device.css
    │  │      nav.css
    │  │
    │  ├─img
    │  │      co2.png
    │  │      EC.png
    │  │
    │  └─js
    └─templates
            device.html
```
* static 静态资源,css/js/图片对应放到css/js/img文件夹
* templates 页面
